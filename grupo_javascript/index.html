Incluir javascript

En este nuevo vídeo vamos a ver como podemos incluir nuestro codigo de javascript dentro de nuestros proyectos. Veremos como inlcuirlo tanto en linea como en archivos separados, siendo esta última una buena práctica de programación.

Variables

Vamos a ver que son y para que se utilizan las variables. Las Variables no son mas que pequeños espacios en memoria donde guardaremos la información necesaria para su posterior utilización.

Constantes

A diferencia de las variables las constantes no pueden variar su valor en tiempo de ejecución, por lo tanto son muy útiles a la hora de definir valores que seran fijos dentro de nuestro codigo.

Datos primitivos

Estos son los datos mas comunes dentro de cualquier lenguaje de programación como pueden ser

  • String
  • Number
  • Boolean
  • Objetc

Tipado dinámico

Esta es una carateristica de javascript que hay que conocer, ya que este lenguaje no consta de un tipado de datos per-se, cada variable puede contener un dato diferente y este en tiempo de ejecución puede ir variando.
